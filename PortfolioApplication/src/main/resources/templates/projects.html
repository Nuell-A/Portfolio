Quiz Game
  • Python
  • Pygame
  • MySQL
  • Socket Programming
  • Multi-threading

This project was created using Python. It implements a client-server model. The clients interact with the UI which in turn sends requests to the server and the clients are able to handle the response from the server and act accordingly. The server handles the application logic. It initiates the connection with the database (MySQL), listens for connections and handles them (can handle multiple connections through multi-threading).

Users can host or join sessions, once a session is joined there is lobby where players can chat real time. Once the host starts the game, players are moved to the quiz state where questions are shown and there is a timer with a score tracker. Lastly, when all questions are done, the results are shown with each player's score.

Book Store
  • Python
  • Flask
  • HTML5&CSS
  • Jinja
  • Bootstrap
  • MySQL
  • Password Encryption

This project was created with Python as a web application. Using the Flask web framework. The UI was created with HTML and Bootstrap CSS along with normal custom CSS. The back-end uses a MySQL server which holds the data for the books and users. MySQL data is managed through sqlalchemy (ORM). Passwords for user accounts are stored securely (hashed).

Users can search books, add books to their cart, checkout their cart, login or create an account, return books, and logout.
